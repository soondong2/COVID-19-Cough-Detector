## 서비스 내용
### COVID-19 검출 표준 방법
- 역전사 중합효소 연쇄 반응(RT-PCR) 검사
- 민감도와 특이도가 가장 높아 전 세계적으로 코로나19 감염의 표준 검사법으로 사용되고 있다.
<br>

**그러나 전용 장비와 시약, 숙련된 전문 인력이 필요하며 비용과 시간이 많이 든다는 단점이 있다.**<br>
